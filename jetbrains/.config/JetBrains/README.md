# JetBrains IDE Settings

JetBrains IDEs have built-in settings sync. This directory holds any manually exported settings.

## Recommended: Settings Sync

1. Open your JetBrains IDE
2. Go to **File > Manage IDE Settings > Settings Sync...**
3. Sign in with your JetBrains account
4. Enable sync for desired categories (code style, keymaps, plugins)

Settings sync automatically across all machines where you are signed in.

## Alternative: Export/Import

1. **Export**: File > Manage IDE Settings > Export Settings... (saves a `.zip`)
2. Place the `.zip` in this directory
3. **Import on another machine**: File > Manage IDE Settings > Import Settings...
